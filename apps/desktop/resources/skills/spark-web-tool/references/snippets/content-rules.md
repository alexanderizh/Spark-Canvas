## 绝对内容规则

### 禁止事项 — 违反将导致输出被拒：
- 元语言预告内容："本场景将介绍..."、"我们今天来学习..."
- 只有公式名称而没有实际公式：写了"牛顿第二定律"却没写 F=ma
- 只有例题标题而没有完整解答："求解示例"但没有任何实际数值
- 要点仅为单个词语或主题标签：["光合作用", "叶绿体"] — 这些是无效的
- 要点列表读起来像目录而非知识内容
- 模糊的视觉描述："展示XX的过程"是禁止的 — 必须指定元素类型和标签

### 每个场景必须包含：
- 旁白/叙述：100–300 字的实际内容，包含具体信息
- 公式/示例类型场景至少包含一个完整的示例
- 要点：完整句子，如"光合作用在叶绿体中进行，分为光反应和暗反应两个阶段"
- 视觉描述：指定图表类型 + 列出所有文字标签 + 配色方案

### 语言规则：
- 与源材料语言保持一致（中文源材料 → 中文输出）
- 技术术语可保留原语言（如英文缩写）
- `math_expressions` 中的 LaTeX 使用标准 LaTeX 语法（无需额外转义）
