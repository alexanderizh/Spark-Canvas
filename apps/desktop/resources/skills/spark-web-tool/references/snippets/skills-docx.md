## 📄 DOCX 文档生成技能指引

### ━━━ 文档结构与排版（核心）━━━

**/minimax-docx** ⭐⭐⭐ — 专业 DOCX 排版
- 用于：多级标题样式、表格边框与底纹、页眉页脚、段落间距控制
- 关键：标题层级最多 3 级；正文字号 12pt，行距 1.5 倍；段落间距 6pt
- 表格规范：表头加粗居中，数据行交替底色，边框统一 0.5pt

**/docx** ⭐⭐ — 文档结构最佳实践
- 用于：封面→目录→序言→主章节→总结→术语表的标准结构
- 防止：标题层级混乱、正文与标题样式混用、缺少目录

**/doc-coauthoring** — 结构化文档协作
- 用于：优化内容的章节组织（先总后分/渐进式）、逻辑流设计

**/document-skills:docx** — DOCX 格式规范
- 用于：DOCX 创建最佳实践和格式规范，补充 minimax-docx 的规范细节

**/document-skills:canvas-design** — 专业文档视觉设计
- 用于：文档中图表、封面、章节标题的视觉设计原则

### ━━━ 图表与可视化━━━

**/vchart-development-assistant** ⭐ — 图表配置
- 用于：生成图表数据，以图片形式嵌入 DOCX
- 场景：数据表格旁附图、知识点统计图

**/excalidraw-diagram-generator** ⭐ — 流程图表
- 用于：生成步骤流程图/知识结构图，导出 PNG 嵌入 DOCX
- 场景：解题流程、知识点关系图、公式推导步骤图

### ━━━ 内容质量━━━

**/brand-guidelines** — 文档配色
- 用于：标题色/强调色/表格色的专业一致性

**/tutor-skills** — 内容结构化
- 用于：将内容组织为"概念→原理→示例→变式→总结"的结构化流程
