## 【字体规范】

- 字体选型与搭配由所选 UI 设计技能（`/ui-ux-pro-max` 或 `/design-taste-frontend`）根据内容调性决定，整个产物保持同一套字体体系
- 中文字体必须提供可靠的系统回退栈（如 `"PingFang SC", "Microsoft YaHei", "Source Han Sans SC", sans-serif`），保证在无自定义字体时仍正常渲染

## 通用要求
- 所有产物写入 output/ 目录
- 不遗漏任何场景
- 如果同时输出多种格式，确保内容一致但详细程度可不同
- **图片生成**：如果系统注入了图片生成能力（见系统提示词"AI 图片生成能力"章节），积极为以下场景生成配图：封面图、内容页背景图（加遮罩）、概念插图、流程/信息图、总结页配图。判断标准：**如果图片能更好地表达内容，就应生成**。不同格式中的使用方式：
  - **HTML**：作为 `background-image`（加半透明遮罩）、`<img>` 内联插图、封面 hero 图
  - **PPTX**：通过 `add_picture()` 嵌入封面全屏背景、章节分隔页、内容插图
  - **DOCX**：通过 docx 库嵌入封面图、章节开头插图
